Arrays

Los arrays son un tipo de datos que no hemos tratado al inicio porque su tratamiento es algo más complejo que los tipos básicos. No obstante su conocimiento y dominio resultan fundamentales para el uso común del lenguaje. Un array es una colección de datos.

En el lenguaje PHP los datos pueden ser del mismo tipo o no, pueden incluso ser otros arrays u objetos.

Según la forma de acceder a cada uno de los datos hablamos de arrays ordeandos u asociativos

Arrays ordenados

Un array ordenado es una colección de datos donde cada uno de ellos es referenciado por la posición que ocupa en la misma.

Los arrays pueden ser inicializados de varias maneras distintas. Veamos:

  • En la declaración de la variable. Los dadtos van entre paréntesis y separados por comas:
    <!DOCTYPE html>
    <html><head>
            <meta charset="UTF-8">
            <title>ejemplo 4</title>
    </head><body>
    <?php

    echo "Array ordenado inicializado en la declaración <hr>";
    $array1 = array(5,"hola", 2.3, true, false, "elemento5");
    //accedemos a los elementos del array con corchetes y su posición.
    $elemento0 = $array1[0];
    echo     "El elemento 1 es $elemento0 y también $array1[0]";
    echo     "<hr> Podemos ver el contenido con var_dump o print_r";
    var_dump($array1);
    print_r($array1); //print_r puede ser más incomodo salvo que usemos <pre></pre>

    echo "<pre>";
    print_r($array1); 
    echo     "</pre>";

    //la podemos recorrer con el bucle foreach de dos maneras:
    echo "<hr> Recorrido mediante bucle foreach simple <br>";
    foreach (    $array1 as $element){
        echo     $element . '<br>';
    }
    //foreach clave->elemento
    echo "<hr> Recorrido mediante bucle foreach con posicion->elemento <br>";
    foreach (    $array1 as $position=>$element){
        echo     $position . ": " . $element . '<br>';
    }
    ?>
    </body></html>

  • En cualquier momento podemos añadir nuevos elementos indicando o no su posición. Internamente se lleva cuenta del último elemento añadido para que los nuevos ocupen las siguientes posiciones.
    <!DOCTYPE html>
    <html><head>

    print_r($array1);

        echo     $element . '<br>';
    }
    //foreach clave->elemento
    echo "<hr> Recorrido mediante bucle foreach con posicion->elemento <br>";
    foreach (    $array1 as $position=>$element){
        echo     $position . ": " . $element . '<br>';

Arrays asociativos

Los arrays asociativos usan una etiqueta en lugar del número de posición.

<!DOCTYPE html>
<html><head>
        <meta charset="UTF-8">
        <title>ejemplo 6</title>
</head><body>
<?php
//inicializamos
echo "Array ordenado inicializado en la declaración <hr>";
$array1 = array (
    'id' => 5,
    'nombre' => 'Manuel',
    'apellido' => 'García López',
    'edad' => 23
);


//foreach clave->elemento
echo "Recorrido mediante bucle foreach con posicion->elemento <br>";
foreach ($array1 as $position=>$element){
    echo $position . ": " . $element . '<br>';
}
echo 'añadimos un elemento más y hacemos un volcado: <br>';
$array1['nacionalidad'] = 'Española';
var_dump($array1);
?>
</body></html>

Arrays de dos dimensiones

Los elementos de un array pueden ser nuevos arrays. Esto permite recoger el contenido de tablas en arrays de dos dimensiones donde cada elemento del array principal es un array.

Los arrays de dos dimensiones pueden ser ordenados, asociativos o una mezcla de ambos

<!DOCTYPE html>
<html><head>
        <meta charset="UTF-8">
        <title>ejemplo 7</title>
</head><body>
<?php
//inicializamos
echo 'Array ordenado doble inicializado en la declaración <hr>';
$ciudades = array (
    'Zaragoza' => array(
        'poblacion' => 700000,
        'gentilicio' => 'Zaragozano'
    ), 
    'Madrid' => array(
        'poblacion' => 3000000,
        'gentilicio' => 'Madrileño'
    ), 
    'Barcelona' => array(
        'poblacion' => 3000000,
        'gentilicio' => 'Barcelonés'
    ), 
    'Bilbao' => array(
        'poblacion' => 500000,
        'gentilicio' => 'Bilbaino'
    ), 
);
?>

Recorrido mediante bucle foreach doble <br>
<table>
<tr><td>Ciudad</td><td>Población</td><td>Gentilicio</td></tr>
<?php
foreach ($ciudades as $ciudad=>$datos){    
    echo '<tr>';
    echo "<td> $ciudad </td>";
    foreach ($datos as $clave=>$dato){
        echo "<td> $dato </td>";
    }
    echo '</tr>';
}
?>
</table>
</body></html>
